제어할 태그 선택하기

선택자 넣는 법 검색

javascript select tag by css selector 구글링

document.querySelector(selectors);

selectors에 'body' 넣으면 바디태그 선택됨


설렉터 뒤에 스타일 넣는 법 검색

javascript element style 구글링

document.getElementById("myH1").style.color = "red";

설렉터 뒤에 .style 넢으면 됨


설렉터 뒤에 스타일 뒤에 배경 넣는 법 검색

javascript style background-color 구글링

document.body.style.backgroundColor = "red";

스타일 뒤에 .backgroundColor = '' 넣으면 됨

아무 성질이 없는 태그

span : 줄바꿈조차 안됨, 아무 특징 없음
