비교 연산자와 블리언

비교 연산자 (Compoarison Operatior) / 불리언 (Boolean)

===

비교 연산자면서, 이항 연산자

좌항과 우항이 같으면 true를 출력, 다르면 false를 출력

여기서 true와 false 두 값 = boolean 이라고 함

document.write(1===1);

document.write(1===2);

<

document.write(1<2);

document.write(1<1);

데이터 타입

숫자열 number

숫자는 무한이 많음, 숫자열의 값도 무한히 많음

문자열 string

문자도 매우 많음(인간의 언어만큼), 문자열의 값도 엄청 많음

불리언 boolean

2개의 데이터만 있음, true/false

이 2개로 인해 -> 인간의 단순 반복 업무를 해방시킴, (영국 수학자 '조지 부울'의 수학적 업적을 기념하기 위한 이름)
