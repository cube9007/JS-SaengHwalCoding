배열

배열이 왜 필요한가?

반복문을 배우면 이해될 것임
살림이 늘어나면 책장이나 서랍을 사게 됨 = 집에 있는 사물이 많아질수록 정리정돈이 필요하게 됨 
코딩도 마찬가지 - 굉장히 많은 데이터를 다루게 됨 = 수납 상자로 단순화할 필요가 있음 
배열 = 데이터를 정리하는 수납상자

배열을 만드는 방법

Syntax = 문법 

문자는 따옴표 "로 시작해서 따옴표로 끝남 
배열은 대괄호 []로 시작해서 대괄호로 끝남
var coworkers = [ 'egoing', 'leezche' ]; 
coworkers라는 수납상자에 2개를 넣어놓은 상태

get = 꺼내오는 법

document.write(coworkers[0]); document.write(coworkers[1]); 
변수 뒤에 [0] = 배열 중 0번째를 가져오겠다 
이 0,1,2,3 = Index라고 부름

count

document.write(coworkers.length); 
배열 변수 뒤에 .length를 붙이면 배열의 갯수를 알 수 있음
