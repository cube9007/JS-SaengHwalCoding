반복문 Loop

반복문은 왜 필요한가?

<ul> <script> document.write('<li>1</li>'); document.write('<li>2</li>'); document.write('<li>3</li>'); document.write('<li>4</li>'); </script> </ul> 
예를 들어 위와 같은 코드를 1억번 반복해야된다면, 복붙으로는 한계가 있음 
하지만 반복문을 쓰면 쉽게 할 수 있음

반복문 예시

<ul> <script> document.write('<li>1</li>'); var i = 0 while(i < 3) { document.write('<li>2</li>'); i = i + 1 } document.write('<li>3</li>'); document.write('<li>4</li>'); </script> </ul> 
2번쨰 li를 3번 반복시키는 반복문임
반복문은 while문을 씀
while뒤에 오는 ()에 불리언(true or false) 를 넣음
관습적으로 변수는 i를 씀
i는 매 실행될때마다 1씩 증가하도록 i = i + 1 을 넣음
while의 ()안에 i는 3보다 작다는 문구를 넣으면 완성
