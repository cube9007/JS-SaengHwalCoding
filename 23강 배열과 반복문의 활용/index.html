배열과 반복문의 활용

일반적인 셀렉터 사용시 문제점

document.queryselector('a'); 
콘솔에서 셀렉터로 a태그를 지정하면 1개만 지정됨 
querySelector라는 명령은 1개만 가져오는 특성을 가지고 있음 
*메소드 = 함수 = 명령 다 같은 말
그래서 다른 걸 찾아봐야됨 검색 ㄱㄱ
javascript get element by css selector multiple 구글링
검색 결과가 querySelectorAll이 나옴 다시 이걸 검색 ㄱㄱ
javascript querySelectorAll 구글링
var matches = document.querySelectorAll("p");

이런 형식이 나옴

var matches = document.querySelectorAll("div.note, div.alert");

해당 태그(div)의 클래스도 정의할 수 있음

var container = document.querySelector("#test");
var matches = container.querySelectorAll("div.highlighted > p");

해당 태그(#, div)의 id값이나 자식 태그도 정의할 수 있음
