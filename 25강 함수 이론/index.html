함수 function (나중에 객체 배우면 메소드 method라고 부르기도 함)

함수의 기본적인 문법

  • 1
  • 2-1
  • 2-2
  • 3
  • 2-1
  • 2-2
<ul> <li>1</li> <li>2-1</li> <li>2-2</li> <li>3</li> <li>2-1</li> <li>2-2</li> </ul> 
만약 반복문을 쓰려고 하는데, 규칙적으로 반복되는게 아니라면
반복문을 사용하기가 어려워지거나 불가능해짐
(중간에 뭐가 하나 껴있다거나, 빠져야된다거나 등등)

함수 사용 예시

<script> function two() { document.write('<li>2-1</li>'); document.write('<li>2-2</li>'); } document.write('<li>1</li>'); two(); document.write('<li>3</li>'); two(); </script> 
똑같아 보이지만 다른 코드를 사용 
function 뒤에 이름을 써줌 ( two라고 짓겠음 )
이름 뒤엔 무조건 ()괄호 쓸 것
함수 내용은 () 뒤의 {}중괄호 안에 넣을 것
