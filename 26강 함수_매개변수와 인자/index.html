함수: 매개변수와 인자

함수 = 자판기 

여태까지 배운 것은 자판기에서 1가지 제품만 나오는 것과 같음 (똑같은 제품만 받음)
편리하긴 하지만 아쉬움
원하는 제품을 선택하면 구체적인 제품을 제공해줬음 좋겠음
제품의 선택 = 입력
자판기의 물건 제공 = 출력
* 이 세상의 모든 것들이 입력과 출력으로 이루어진 것이라고 보아도 무방함 
* 입력과 출력에 해당되는 것이 수학적으로 함수라고 볼 수 있음 

함수 = 입력과 출력으로 이루어짐
입력 = 파라미터 (parameter, 매개변수)
    = 아규먼트 (argument, 인자) 
출력 = 리턴 (return)

함수의 파라미터(매개변수)와 아규먼트(인자)의 예

function naming(left, right) { <- 파라미터 (매개 변수) : 인자를 가지고 와서 매개시킴(관계를 맺어줌) document.write(left+right+'<br>') } naming(2,3); <- 아규먼트 (인자) : 함수로 전달하는 2,3이라는 값 
* 매개 = 둘 사이에서 관계를 맺어줌
파라미터, 아규먼트 두 가지는 함수로 들어오는 입력값과 관련이 있음
