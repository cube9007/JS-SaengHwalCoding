함수 리턴

함수 리턴을 하기 전, 표현식이란 무엇인가? (expression)

1+1 2 2-1 1 1 === 1 - true라는 값에 대한 표현식 true

리턴 활용

<script> function sum2(left,right) { - left,right = 매개변수 (파라미터) return left + right ; - 리턴구문 } document.write(sum2(2,3)+'<br>'); - 2,3 = 인자 (아규먼트) document.write('<div style="color:red">' + sum2(4,5)+'</div>'); - 4,5 = 인자 (아규먼트) document.write('<div style="font-size:25px">' + sum2(3,4)+'</div>'); - 3,4 = 인자 (아규먼트) </script> 
아규먼트의 값을 파라미터에 대입해서, 리턴구문에 표현식으로 넣음
