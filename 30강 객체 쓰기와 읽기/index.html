객체 쓰기와 읽기 object

객체란? 

배열은 순서대로 정보를 정리하는 것이라면
객체는 순서 없이 정보를 정리하는 것 - 그렇다고 무작위로 넣는 것은 아님 (그건 쓰레기통)
- 쓰레기통 = 꺼낼 필요가 없음
- 객체 = 꺼낼 필요가 있음 ( 서로 다르다 )

배열은 이름 없이 순서대로 넣지만 - 순서로 꺼냄
객체는 이름을 같이 적어 순서 없이 넣음 - 이름으로 꺼냄

읽고 쓰기

<script> var naming = { 'designer' : 'wonho', 'publisher' : 'minsu' } document.write('designer : ' + naming.designer + '<br>'); document.write('publisher : ' + naming.publisher + '<br>'); naming.planner = 'yuri'; document.write('planner : ' + naming.planner + '<br>'); </script> 
배열은 [대괄호]를 쓰지만
객체는 {중괄호}를 씀
중괄호 안에는 
    { 객체의 요소 이름 : 객체의 대상, }
중괄호 밖에서 객체를 추가할 떄는
    객체이름.객체요소의 이름 = '객체 대상';
