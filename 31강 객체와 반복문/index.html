객체와 반복문

반복문을 통해 데이터 가져오기 

배열은 반복문을 통해 데이터를 다 가져옴
객체도 반복문을 통해 데이터를 다 가져올 수 있음 
* key 값 : 객체에서 꺼내오는 대상에 붙인 이름표의 값 
* index 값 : 배열에서 꺼내오는 대상의 순서 값

실습 for문 (var key in object) {}

객체 반복문 검색 javascript object iterlate 
for (const property in object) {
    console.log(`${property}: ${object[property]}`);
    }
for(var key in naming) { document.write(key + ' : ' + naming[key] + '<br>'); } 
for문 안에 ()에 var key in naming 을 적어놓음
naming 안에 있는 key 값들을 가져오는 for문(반복문)
